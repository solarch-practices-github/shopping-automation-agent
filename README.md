This project aims to automate online shopping tasks using AI agents.
It's built with Claude Agent SDK for nodejs.


## References

- Follow [this](https://platform.claude.com/docs/en/agent-sdk/overview.md) page and links within to learn more about Claude Agent SDK and how to build agents using it.

- Read [this](https://platform.claude.com/docs/en/about-claude/models/overview) page to understand names and pricing of current Anthropic Claude models.

- Check out [this](https://raw.githubusercontent.com/microsoft/playwright-mcp/refs/heads/main/README.md) page to learn more about Playwright MCP its tools and config options.

## Run locally

Create `.env` with the following variables:

```
ANTHROPIC_API_KEY=your_anthropic_api_key_here
```

Check `playwright-mcp.json`: make sure browser path is correct for your system.
